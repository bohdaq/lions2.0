# Lions FC - Sports Club Landing Page

A modern, one-page landing website for a sports club inspired by FC Karpaty Lviv's design aesthetic.

## Features

- **Bold Hero Section** - Eye-catching hero with club name, tagline, and call-to-action buttons
- **Clean Navigation** - Fixed top navigation bar with smooth scrolling
- **About Section** - Club information with statistics showcase
- **Team Section** - Player cards with jersey numbers and positions
- **Latest News** - News grid with featured article layout
- **Media Gallery** - Photo and video sections with tabbed interface
- **Partners Section** - Partner/sponsor logo grid
- **Contact Section** - Contact information and functional contact form
- **Footer** - Comprehensive footer with links and newsletter signup

## Design Elements

- **Athletic Style** - Bold typography using Bebas Neue and Montserrat fonts
- **Club Colors** - Gold (#D4AF37) and black color scheme
- **Dynamic Animations** - Smooth transitions, hover effects, and scroll animations
- **Responsive Design** - Fully responsive layout for all device sizes
- **Modern UI** - Clean, professional design with strong visual hierarchy

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Bebas Neue, Montserrat)

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required
3. All assets are self-contained

## Customization

- **Colors**: Edit CSS variables in `:root` selector in `styles.css`
- **Content**: Update text, images, and information in `index.html`
- **Fonts**: Modify Google Fonts imports in HTML head section
- **Sections**: Add or remove sections as needed
